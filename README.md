# React Step Form

> Form with multiple steps and confirmation. Frontend only, using API. Uses Material UI

## Quick Start

```bash
# Install dependencies
npm install

# Serve on localhost:3000
npm start

# Build for production
npm run build
```

## App Info
React js
Bootstrap
React Select
